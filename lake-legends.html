Lake Legends

Mar 2021 – Present

Description

Lake Legends is a fantasy fishing game focused on exploration, collection, and progression. Players travel across mystical lakes inspired by Viking, Norse, and medieval themes, catching more than 500 fish species while uncovering environmental lore and hidden stories tied to each location.
The game includes gear collection and upgrade systems, including a gacha-based mechanic to earn fishing gear. As a free-to-play title, Lake Legends combines relaxing fishing mechanics with long-term progression, collectible content, and light narrative exploration.
It was originally a month long student project at Forsbergsskola that eventually became a commercial project when we started Bridge Games Studio.

Role

Lead developer. Responsible for developing the entire game, including gameplay mechanics, loot generation, integrating plugins, and gameplay architecture.

Contributions

  • Integrated Google Firebase for NoSQL database, account management, analytics, and crashlytics.
  • Created an extensive loot system with thousands of variations.
  • Created multiple gameplay systems such as achievements, daily objectives, tutorials, and the main fishing minigame.
  • Leveling system with experience points, rewards, and unlockable profile pictures.
  • Implemented Unity LevelPlay Ads for rewarded ads.
  • Implemented Unity In-App Purchasing for in-game purchases.
  • Implemented Unity Remote Config for dynamic game settings.
  • GitHub Actions CI/CD for Android and iOS running on a self-hosted macOS runner
